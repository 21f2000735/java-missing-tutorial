# Top 20 Book Resources

This is a curated list for building this repo into a serious Java learning resource.

It is not a popularity ranking. It is a coverage list based on core Java, OCJP preparation, testing, concurrency, design, and modern Java practice.

## Core Java

1. Effective Java, 3rd Edition, Joshua Bloch
   Source: https://www.informit.com/store/effective-java-9780134686042
   Use for: design rules, API design, generics, concurrency, exceptions, and best practices.

2. Core Java, Volume I: Fundamentals, 14th Edition, Cay S. Horstmann
   Source: https://www.informit.com/store/core-java-volume-i-fundamentals-9780135558577
   Use for: strong examples for language basics, OOP, generics, collections, lambdas, and concurrency.

3. Core Java, Volume II: Advanced Features, 14th Edition, Cay S. Horstmann
   Source: https://www.informit.com/store/core-java-volume-ii-advanced-features-9780135558690
   Use for: streams, IO, networking, JDBC, date/time, internationalization, and advanced APIs.

4. Modern Java in Action, Raoul-Gabriel Urma, Mario Fusco, Alan Mycroft
   Source: https://www.manning.com/books/modern-java-in-action
   Use for: lambdas, streams, modules, and modern Java thinking.

5. Learn Java 17 Programming, 2nd Edition, Nick Samoylov
   Source: https://www.packtpub.com/en-us/product/learn-java-17-programming-second-edition-9781803241432
   Use for: beginner-friendly progression and broad chapter scaffolding.

6. Data-Oriented Programming in Java, Chris Kiehl
   Source: https://www.manning.com/books/data-oriented-programming-in-java
   Use for: records, sealed types, pattern matching, and modern domain modeling.

## Concurrency and Reactive Systems

7. Java Concurrency in Practice, Brian Goetz et al.
   Source: https://www.informit.com/store/java-concurrency-in-practice-9780321349606
   Use for: thread safety, synchronization, concurrency design, and mental models.

8. Functional Programming in Java, Pierre-Yves Saumont
   Source: https://www.manning.com/books/functional-programming-in-java
   Use for: immutable thinking, functional style, and stream-friendly design.

9. Reactive Design Patterns, Roland Kuhn, Brian Hanafee, Jamie Allen
   Source: https://www.manning.com/books/reactive-design-patterns
   Use for: concurrency architecture, responsiveness, resilience, and resource control.

10. Vert.x in Action, Julien Ponge
    Source: https://www.manning.com/books/vertx-in-action
    Use for: asynchronous and event-driven Java systems.

11. Microservices Patterns, Chris Richardson
    Source: https://www.manning.com/books/microservices-patterns
    Use for: service design, transactions, boundaries, and production patterns.

## Testing, Clean Code, and Refactoring

12. Unit Testing: Principles, Practices, and Patterns, Vladimir Khorikov
    Source: https://www.manning.com/books/unit-testing
    Use for: chapter testing strategy, examples, and anti-pattern detection.

13. Unit Testing with Java, JUnit 5, and IntelliJ IDEA, Max Guernsey
    Source: https://www.manning.com/liveprojectseries/unit-testing-with-Java-JUnit-5-and-IntelliJ-IDEA
    Use for: practical JUnit workflow and teaching exercises.

14. Clean Code, 2nd Edition, Robert C. Martin
    Source: https://www.informit.com/store/clean-code-a-handbook-of-agile-software-craftsmanship-9780135398579
    Use for: readability, naming, function design, and code style rules.

15. Refactoring, 2nd Edition, Martin Fowler
    Source: https://www.informit.com/store/refactoring-improving-the-design-of-existing-code-9780134757698
    Use for: before/after examples and code-improvement chapters.

16. Clean Architecture, Robert C. Martin
    Source: https://www.informit.com/store/clean-architecture-a-craftsmans-guide-to-software-structure-9780134494166
    Use for: package organization, chapter grouping, and long-term project structure.

17. Troubleshooting Java, Second Edition, Laurentiu Spilca
    Source: https://www.manning.com/books/troubleshooting-java-second-edition
    Use for: debugging chapters, code reading, and production investigation skills.

## Interview and Ecosystem

18. The Complete Coding Interview Guide in Java, Anghel Leonard
    Source: https://subscription.packtpub.com/book/programming/9781839212062/pref/preflvl1sec01/preface
    Use for: interview-style drills, coding problems, and explanation patterns.

19. Spring Start Here, Laurentiu Spilca
    Source: https://www.manning.com/books/spring-start-here
    Use for: later ecosystem chapters and real-world Java application context.

20. Java Persistence with Hibernate, Second Edition, Christian Bauer, Gavin King, Gary Gregory
    Source: https://www.manning.com/books/java-persistence-with-hibernate-second-edition
    Use for: persistence, ORM, and data access chapters.

## How To Use These Books

- use them as guidance, not as copy sources
- extract ideas for chapter ordering, pitfalls, and better examples
- write original explanations in simpler language
- connect each topic to interview questions, OCJP traps, and Effective Java items
