# Interview Topic Template

Use this template for topics whose main job is to make the learner interview-ready.

Keep the topic short and focused. The default shape is only:

1. topic/problem
2. intuition
3. small code snippet
4. internal working
5. comparison with other options
6. famous company interview question

## Required `TopicGuide.md` Shape

```md
---
introduced: Java 8
status: stable
runner: embedded
estimated: 8 min
mode: interview
visual: recommended
---

# Topic Name

## Topic / Problem
- what the topic is
- what real problem it solves

## Intuition
- the simplest way to think about it
- the mental picture before code

## Small Code Snippet
- one runnable snippet
- what to notice in the output

## Internal Working
- what Java does under the hood
- what happens at compile time or runtime

## Comparison With Other
- compare against the closest alternative
- say when to use each one

## Famous Company Interview Question
- one common company-style question
- a short answer the learner can say out loud
```

Do not add extra headings unless they introduce new information.
