# Interview Track

Use this path if your goal is backend interviews, debugging rounds, and company-style Java screening.

## Why This Track Exists

Interview prep is not the same as certification prep.

You need:

- problem-framing
- tradeoff explanation
- runnable examples you can narrate
- enough JVM, collections, concurrency, and system-boundary understanding to answer follow-up questions

## Practice Loop

1. Pick a topic.
2. Explain the problem in plain English.
3. Run the example.
4. Explain the output.
5. Say when you would use it in production.
6. Say one common mistake and one tradeoff.
